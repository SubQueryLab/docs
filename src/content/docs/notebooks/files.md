---
title: Работа с файлами
---

Ноутбук поддерживает работу с локальными файлами.

## Что это дает

- загружать файлы для анализа без отдельного ETL;
- быстро проверять CSV/Excel/Parquet перед запросами к СУБД;
- комбинировать данные из файлов и результаты SQL-запросов.

## Как обычно строится процесс

1. Добавьте файл в ноутбук через сайбар или dnd.
2. Откройте его в ячейке `python` или `duckdb`.
3. Продолжите обработку в следующих ячейках.

## Пример в Python

```python
import pandas as pd

pd.read_csv("filename.csv")
```

## Пример с DuckDB

```sql
select * from sq_data.filename
```

Все загруженные файлы доступны в DuckDB через схему sq_data как view с именем файла

## Важно

Загруженные файлы могут сохраняться в сервисе при завершении сессии. 
Для этого необходимо дать разрешение в окне при загрузке
